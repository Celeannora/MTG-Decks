# MTG Deck Lists

This folder contains AI-generated Magic: The Gathering deck lists.

## Format

Decks are saved in standard MTG deck list format:

```
// Deck Name
// Format: Standard
// Generated: [date]

// Creatures (X)
4 Card Name
3 Another Card

// Spells (X)
4 Instant
2 Sorcery

// Lands (X)
4 Land Name
...
```

## Organization

Decks are named with the format: `[Archetype]_[Colors]_[Date].txt`

Example: `Aggro_RW_2026-03-07.txt`

## Usage

These deck lists can be:
- Imported into MTG Arena
- Used with deck building tools
- Modified and tested
- Shared with others
